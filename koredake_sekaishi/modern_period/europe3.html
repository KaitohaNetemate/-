フランス革命

　ここではフランス革命についてみていきます。フランス革命は細かく学習しようとするとかなり時間がかかってしまうため、まずは大まかに学習するのが良いと思います。

　まず、フランスではアンシャン=レジームと呼ばれる身分社会が存在していました。具体的には、第一身分が聖職者、第二身分が貴族、第三身分が平民でした。
　当時のフランスは経済的不況に陥っていたため、三部会を開き、聖職者や貴族に対しても課税を行おうとしました。しかし聖職者、貴族と平民との間で議決方法をめぐって対立したため、平民が独自に国民議会を創設し、球技場(テニスコート)の誓いで憲法制定まで国民議会を解散しないことを誓いました。

Jacques-Louis David, Public domain, via Wikimedia Commons
球技場(テニスコート)の誓い

　さらに、パリの民衆が蜂起し、バスティーユ監獄を襲撃し、フランス革命が始まりました。ただし、農民蜂起も発生し、収拾がつかなくなってしまったため、国民議会は封建的特権の廃止を発表しました。一方で、封建地代に関しては有償での廃止であったことに注意しましょう。さらに、人権宣言によって自由、平等というフランス革命での理念を打ち出していきます。

Jean-Jacques-François Le Barbier, Public domain, via Wikimedia Commons
人権宣言

　国民議会は、立憲君主制や財産資格選挙などを規定した1791年憲法を発布し、国民議会の目的であった憲法制定を達成できたため解散します。その後に誕生した立法議会では穏健共和派であるジロンド派によって議会が掌握されます。
　ジロンド派は共和政国家を樹立するため、王政復古を要求していたオーストリアに宣戦布告します。当初は劣勢でしたが、各地から義勇兵が集まってきたたので、ヴァルミーの戦いで勝利することができました。

Horace Vernet, Public domain, via Wikimedia Commons
ヴァルミーの戦い

　その後男性普通選挙(21歳以上の男性全てに選挙権がある形式)によって成立した国民公会によってルイ１６世が処刑され共和政が実現しました。以降のフランスの政治体制を第一共和政といいます。しかし、あまりにも急進的な革命に諸国は反発し、イギリスのピット首相によって対仏大同盟が結成され、西欧諸国がフランスと敵対します。
　こうした状況をうけ国民公会では、急進的共和派であるジャコバン派が主流となり、ロベス＝ピエールが独裁を行います。一方で農民に対しても配慮を行い、封建地代を無償で廃止しました。

Unidentified painter, CC0, via Wikimedia Commons
ロベスピエール

　しかし、ロベスピエールの独裁に反発した者たちがテルミドール9日のクーデタを起こし、ロベスピエールは失脚し総裁政府が成立しました。しかし、バブーフの陰謀というクーデタ未遂が発生するなど、総裁政府の支配は不安定なものでした。
　このような流れの中で、ナポレオン＝ボナパルトがブリュメール18日のクーデタで総裁政府を打倒し、統領政府を樹立しました。

Jacques-Louis David, Public domain, via Wikimedia Commons
ナポレオン＝ボナパルト

　その後、ナポレオンは国民投票で皇帝に即位します。以降のフランスの政体を第一帝政といいます。ナポレオンはアウステルリッツの三帝会戦でロシアとオーストリアを破ることに成功しました。また、ナポレオンは神聖ローマ帝国を滅ぼすことに成功し、当該地域にライン同盟を成立させました。
　しかし、ロシア遠征に失敗したことでフランスは劣勢になります。まず、ライプツィヒの戦いで諸国の連合軍に敗北すると、ナポレオンはエルバ島に流刑とされました。しかしナポレオンはエルバ島から脱出しフランスに戻ることに成功しますが、ワーテルローの戦いで再び敗北し、今度はセントヘレナ島に流刑となり、そのままナポレオンはそこで死亡しました。
　その後、次の章で述べるようにウィーン体制による国際体制が成立します。
